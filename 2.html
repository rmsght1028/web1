CSS이란 무엇인가?

Cascading Style SheetsorCascading Style Sheets (CSS) are languages that described how markup languages are actuallydisplayed in [1], HTML, and XHTML,and can also be used in XML. It is the standard for W3Cand hashigh degrees of freedom when defining layouts and styles. If the markuplanguage (ex: HTML) isresponsible for the body of the website, CSSplays a role in decorating like clothes and accessories. In other words, you can change the CSS file without changing the HTML structureto make it look like a completely different website. CSS3, which iscurrently in development, adds the ability to replace images created bygraphics editing programs, such as shadow effects, gradients, and transformations. In addition, various animation features have been added toreplace Adobe Flash to some extent.
